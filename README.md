# 🏨 Hotel Management System in C++

A fully-featured console-based hotel management system built using C++. This project allows managing room bookings, customer check-ins/check-outs, billing (with food menu), and persistent storage using file I/O.

---

## 🚀 Features

- ✅ Login system (with masked password input)
- ✅ Add and search rooms
- ✅ Check-in and Check-out customers
- ✅ Track available rooms
- ✅ Guest summary report
- ✅ Food ordering and billing
- ✅ File-based data persistence
- ✅ Custom console styling and text animation

---

## 📂 Sections of the Hotel Managed

- 🔹 Room Management
- 🔹 Customer Check-in & Check-out
- 🔹 Billing (Room Rent + Food Orders)
- 🔹 Guest Summary & Reports
- 🔹 File-based persistent storage

---

## 🛠️ Technologies Used

- C++
- File Handling (`fstream`)
- Windows API (`windows.h` for console color)
- `_getch()` for secure input (password masking)
- Text-based UI (no GUI)

---

## 💡 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/Hotel-Management-System-Cpp.git
````

2. Open in any C++ IDE (e.g., Code::Blocks, Dev C++, Visual Studio).
3. Compile and run `main.cpp`.

📌 Make sure you are on Windows, as this uses `windows.h` and `_getch()`.
